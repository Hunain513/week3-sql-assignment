# week3-sql-assignment
In this assignment , we were asked to use MySql and xamp to create a database. In this assignment, I started by creating my own students table using the CREATE TABLE statement and then inserted the given records using INSERT INTO. After making sure the data was added correctly, I completed the easy queries. In the easy section, I practiced basic SQL commands such as SELECT, WHERE, DISTINCT, LIKE, ORDER BY, LIMIT, COUNT(), AVG(), MAX(), MIN(), and IS NULL. These queries helped me understand how to retrieve, filter, sort, and summarize data from a single table.
For the medium section, I worked with the provided employees and departments tables. I used different types of JOINs to combine data from both tables, including INNER JOIN and LEFT JOIN. I also used GROUP BY to group employees by department, HAVING to filter grouped results, subqueries to compare employee salaries with average salaries, EXISTS to find managers, and IFNULL() to treat missing bonuses as zero. I also filtered records using dates and calculated values such as total pay.
In the hard section, I learned more advanced SQL concepts. I used window functions like ROW_NUMBER(), RANK(), and LEAD() to rank employees and compare salaries. I also used a Common Table Expression (CTE) with WITH, combined results using UNION, wrote correlated subqueries to compare salaries within each department, and used EXISTS to find employees managing others. Overall, this assignment helped me understand SQL from basic queries to advanced techniques and improved my confidence in working with databases.
